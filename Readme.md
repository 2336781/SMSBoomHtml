<h2>如何使用？</h2>
很简单，打开index.html，输入暗号<code>NmSl</code>（防止滥用，区分大小写），就行了～<br>
也有<a href="https://hirgen.github.io/">演示网站</a>(不经常更新）<br>
本网页具有极大开放性，您是可以自己添加接口的（手机号用<code>{phone_number}</code>代替）写入js变量<br>
接口可以用抓包工具获取。
